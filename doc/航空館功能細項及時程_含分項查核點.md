## 一、Avatar 影像辨識與紙娃娃系統

**對應工程範圍**

- Unity App 建置、QRCode 拍照功能整合
- 本地影像處理與服裝特徵分析
- 紙娃娃系統（模型分件、組裝、材質套用）
- 手機端與 Server 玩家資料儲存

**規劃**

| 編碼   | 任務名稱                                    |
| ------ | ------------------------------------------- |
| C1-1-1 | 基礎影像擷取與人像辨識模組建置              |
| C1-1-2 | 基礎外觀特徵分類（如髮型、上衣 / 褲子顏色） |
| C1-1-3 | 特徵標註規範與基礎資料集建立                |

**開發**

| 編碼   | 任務名稱                                                   |
| ------ | ---------------------------------------------------------- |
| C1-2-1 | 辨識模型擴充，特徵分析眼睛大小、髮長、是否配戴眼鏡／帽子等 |
| C1-2-2 | 建立角色模組資料庫（模型元件庫）                           |
| C1-2-3 | 整合角色生成與玩家個人化 App （ID、角色、遊戲進度更新）    |
| C1-2-4 | App 顯示遊玩進度紀錄功能                                   |

### 使用者流程圖：Avatar 影像辨識與紙娃娃系統

```mermaid
flowchart TD
    A[開始] --> B[玩家開啟 App 進入掃描個資頁面]
    B --> C[依指示使用攝像機自拍]
    C --> D[系統進行影像分析]
    D --> E{辨識成功嗎?}
    E -->|否| F[提示重新拍攝或重新辨識]
    F --> C
    E -->|是| G[分析服裝與外觀特徵]
    G --> H[套用紙娃娃模型與配件]
    H --> I[顯示 3D 預覽結果]
    I --> J[玩家確認並送出]
    J --> K[玩家資料回傳 Server]
    K --> L[供後續各遊戲使用]
```

### 系統時序圖概況：影像辨識 → Avatar

```mermaid
sequenceDiagram
    participant P as 玩家
    participant App as App
    participant S as Game Server

    P->>App: 1. 取得 App，玩前用攝像機自拍
    App->>App: 2. 影像分析（工程判斷）
    App->>App: 3. 服裝分析與紙娃娃搭配判斷
    App->>App: 4. 組裝、顯示 3D 預覽（自拍變 3D 娃娃）
    App->>S: 5. 玩家資料回傳 API
    S->>S: 6. 記錄，供各遊戲使用
```
