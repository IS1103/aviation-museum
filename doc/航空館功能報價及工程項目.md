## 二、企劃建議和落地方法

1. 玩家可以乘坐自己塗鴉的飛行器在多人遊戲裡面玩，玩家會增加流暢且完整的遊玩體驗，不加價。
2. 因為手機型號都不同，可能會讓分析精准度下降，由一台裝有攝像機的螢幕負責半身攝像、儲存，玩家手機負責掃描 qecode 登陸資訊，影像辨識的速度比較可控，玩家操作也會比較直觀方便，這裡會需要再做新的 Unity 場景但不加價。
3. Text-to-Image 換思路解法：從原本 AI 生模型改成 AI 取得語音文字後，分析並取得最接近目前已經做好的飛行器白模，並回傳 Client 調用該模型做塗鴉，這樣好做，查核點風險也降低。
4. 2D 貼圖轉 3D 模型的解法：讓模型都做左右對稱，2D塗鴉做特殊設計只需要畫一邊，另一邊做鏡像貼圖再裝到3D模組即可完美呈現3D模型。
5. 多人遊戲玩法流程我可以協助規劃。

## 三、玩家遊戲流程

### 3.1 影像辨識 → Avatar

```mermaid
sequenceDiagram
    participant P as 玩家
    participant App as App
    participant S as Game Server

    P->>App: 1. 取得 App，玩前用攝像機自拍
    App->>App: 2. 影像分析（工程判斷）
    App->>App: 3. 服裝分析與紙娃娃搭配判斷
    App->>App: 4. 組裝、顯示 3D 預覽（自拍變 3D 娃娃）
    App->>S: 5. 玩家資料回傳 API
    S->>S: 6. 記錄，供各遊戲使用
```

### 3.2 塗鴉轉 3D 並投影在畫布

```mermaid
sequenceDiagram
    participant P as 玩家
    participant App as App
    participant AI as AI
    participant S as Game Server
    participant Proj as 投影 Client

    P->>App: 1. 掃螢幕 QR 進入遊戲
    P->>App: 2. 用說的指定想要的飛行器
    App->>App: 3. 語音矯正
    App->>AI: 4. 語音分析請求
    AI->>AI: 5. 分析、簡化、匹配飛行器資料庫
    AI->>App: 6. 回傳飛行器 txt
    App->>P: 7. 顯示對應線稿圖供塗鴉（對應 3D 模型）
    P->>App: 8. 上色塗鴉完成
    App->>App: 9. 2D 對稱貼圖 + Normal 固定 + Albedo 玩家
    App->>S: 10. 上傳飛行器、記錄
    S->>Proj: 11. 轉發玩家飛行器
    Proj->>Proj: 12. 投影畫布上顯示
```

### 3.3 多人連線遊戲

```mermaid
sequenceDiagram
    participant P as 玩家
    participant App as App
    participant S as Game Server
    participant Proj as 投影 Client

    P->>App: 1. 掃螢幕 QR 進入遊戲
    App->>S: 2. 取得玩家資料
    S->>Proj: 3. 推送玩家資料（模型、頭像、飛機等）
    Proj->>Proj: 4. 顯示玩家模型、頭像、自畫飛行器；手機變手把
    P->>App: 5. 操作手把
    App->>S: 6. 控制同步
    S->>Proj: 7. 同步至投影畫面
    Proj->>Proj: 8. 遊戲進行
    Proj->>S: 9. 遊戲結束，回傳結果
    S->>S: 10. 儲存遊戲結果
```

### 3.4 短影音

```mermaid
sequenceDiagram
    participant P as 玩家
    participant App as App
    participant S as Game Server
    participant Third as Third party

    P->>App: 1. 將遊戲記錄變短影音
    App->>S: 2. 取得遊戲記錄
    S->>App: 3. 回傳記錄
    App->>App: 4. 制式美術模板排版
    App->>App: 5. 生成短片、儲存
    App->>Third: 6. 呼叫 Third party 上傳/發布
    Third->>App: 7. 回傳影片連結
    App->>P: 8. 分享連結
```
